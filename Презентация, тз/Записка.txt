В приложении на данный момент есть 12 героев, для каждого из них сделаны свой ui файл,
описание их способностей, сборки самих способностей, предметов, а также их лор (биография) и противодействия.
В главном меню есть кнопка смены фона, имя спонсора, ссылка для пожертвования (<3), а также логотип нашей команды.
Есть поиск, работающий при нажатии пробела, который по базы данных ищет информацию.
